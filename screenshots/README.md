## 🎨 设计规范

### 配色系统
- **主色**：森林深绿 #2D5F3F
- **深色**：深林绿 #1F4529
- **背景**：白色 #FFFFFF、浅灰 #F9FAFB
- **文字**：深灰 #1F2937、中灰 #6B7280

### 图标系统
- **Emoji 图标**：🌱（免费版）、🌲（专业版）、👑（大师版）
- **功能图标**：📷（相机）、🎨（滤镜）、📊（参数）、👤（机位）
- **状态图标**：✓（完成）、›（箭头）、🔒（锁定）

### 圆角规范
- **卡片**：24-32rpx
- **按钮**：48rpx（完全圆角）
- **徽章**：9999rpx（胶囊形）

### 阴影规范
- **标准卡片**：0 4rpx 16rpx rgba(0, 0, 0, 0.06)
- **悬浮卡片**：0 8rpx 32rpx rgba(0, 0, 0, 0.08)
- **按钮**：0 8rpx 24rpx rgba(45, 95, 63, 0.3)
